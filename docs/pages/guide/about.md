# Agent JS

Agent JS is an agent-based modeling library in JavaScript for educational purposes. It is inspired by the [Turtle module of Python][turtle] and [NetLogo][netlogo].

[turtle]: https://docs.python.org/3/library/turtle.html
[netlogo]: https://www.netlogo.org/
